## 底层 SCPI 命令（共 28 条）

来源于 `CViaviPCTDriver.cpp` 中的 `SCPI` 命名空间，参考 MAP-PCT Programming Guide Chapter 5：

### IEEE 488.2 通用命令

| SCPI 命令 | 方向 | 说明                                |
| :-------- | :--- | :---------------------------------- |
| `*REM`    | 写   | 激活远程控制模式（连接后首先发送）  |
| `*IDN?`   | 查询 | 设备标识（制造商/型号/序列号/版本） |
| `*RST`    | 写   | 重置设备到出厂状态                  |
| `*CLS`    | 写   | 清除错误队列和状态寄存器            |
| `*OPC?`   | 查询 | 操作完成查询（返回 1 表示完成）     |

### 系统命令

| SCPI 命令    | 方向 | 说明                                                         |
| :----------- | :--- | :----------------------------------------------------------- |
| `:SYST:ERR?` | 查询 | 查询错误队列。返回 `<code>,"<message>"`，`0,"No error"` 表示无错误 |

### 设备信息

| SCPI 命令 | 方向 | 说明                                                         |
| :-------- | :--- | :----------------------------------------------------------- |
| `:INFO?`  | 查询 | 模块详细信息。返回: `serialNumber,partNumber,fwVersion,modVer1,modVer2,hwVer,date` |
| `:CONF?`  | 查询 | 模块配置。返回: `deviceNum,type,fiberType,outputs,sw1Channels,sw2Channels,wav1,wav2,...` |
| `:BUSY?`  | 查询 | 忙碌状态查询                                                 |

### 光源/波长配置 (Chapter 5.5)

| SCPI 命令                  | 方向 | 说明                                         |
| :------------------------- | :--- | :------------------------------------------- |
| `:SOUR:WAV <nm>`           | 写   | 设置单个波长（nm），如 `:SOUR:WAV 1310`      |
| `:SOUR:WAV?`               | 查询 | 查询当前波长                                 |
| `:SOUR:WAV:AVA?`           | 查询 | 查询可用波长列表（逗号分隔，如 `1310,1550`） |
| `:SOUR:LIST <w1>,<w2>,...` | 写   | 配置多波长列表，如 `:SOUR:LIST 1310,1550`    |
| `:SOUR:LIST?`              | 查询 | 查询当前波长列表                             |

### 路径/通道 (Chapter 5.6)

| SCPI 命令                      | 方向 | 说明                              |
| :----------------------------- | :--- | :-------------------------------- |
| `:PATH:CHAN <group>,<channel>` | 写   | 设置通道路径，如 `:PATH:CHAN 1,3` |
| `:PATH:CHAN?`                  | 查询 | 查询当前通道路径                  |

### 传感器/测量配置 (Chapter 5.7)

| SCPI 命令             | 方向 | 说明                             |
| :-------------------- | :--- | :------------------------------- |
| `:SENS:FUNC <0|1>`    | 写   | 设置测量功能：0=参考, 1=DUT 测量 |
| `:SENS:FUNC?`         | 查询 | 查询当前测量功能                 |
| `:SENS:AVG <seconds>` | 写   | 设置平均时间（秒）               |
| `:SENS:AVG?`          | 查询 | 查询当前平均时间                 |
| `:SENS:RANG <meters>` | 写   | 设置 DUT 范围（米）              |
| `:SENS:RANG?`         | 查询 | 查询当前 DUT 范围                |
| `:SENS:ILON <0|1>`    | 写   | 仅 IL 模式：0=禁用, 1=启用       |
| `:SENS:ILON?`         | 查询 | 查询 IL-only 模式状态            |
| `:SENS:CONN <1|2>`    | 写   | 连接模式：1=单跳线, 2=双跳线     |
| `:SENS:CONN?`         | 查询 | 查询连接模式                     |
| `:SENS:BIDI <0|1>`    | 写   | 双向模式：0=禁用, 1=启用         |
| `:SENS:BIDI?`         | 查询 | 查询双向模式状态                 |

### 测量控制 (Chapter 5.8)

| SCPI 命令     | 方向 | 说明                                                       |
| :------------ | :--- | :--------------------------------------------------------- |
| `:MEAS:STAR`  | 写   | 开始测量（参考或 DUT，取决于 `:SENS:FUNC` 设置）           |
| `:MEAS:STAT?` | 查询 | 查询测量状态：0=初始化, 1=空闲, 2=忙碌, 3=故障, 4=系统功能 |
| `:MEAS:ABOR`  | 写   | 中止当前测量                                               |

### 测量结果 (Chapter 5.9)

| SCPI 命令              | 方向 | 说明                                                         |
| :--------------------- | :--- | :----------------------------------------------------------- |
| `:MEAS:ALL? <channel>` | 查询 | 获取指定通道全部结果。返回: `wavelength,IL,ORL,ORL_zone1,ORL_zone2,length,power,...` |
| `:MEAS:IL? <channel>`  | 查询 | 获取指定通道的插入损耗 (dB)                                  |
| `:MEAS:ORL? <channel>` | 查询 | 获取指定通道的回波损耗 (dB)                                  |
| `:MEAS:LEN? <channel>` | 查询 | 获取指定通道的 DUT 长度 (m)                                  |
